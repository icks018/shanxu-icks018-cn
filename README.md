# 流沙聚·闪讯

> **我们替你在亿万沙粒里筛出真金，60 秒读完世界刚发生的事**

**不堆叠、不注水，只给最锋利的信号**

一个基于热点新闻聚合的内部推送工具，专注于每日热点信息的精准筛选和推送。

## 📋 项目说明

**版本**：v1.0.0 - 流沙聚·闪讯

**功能**：从多个热点平台聚合新闻，通过关键词筛选后推送到指定渠道，适用于内部团队的每日热点信息获取。
